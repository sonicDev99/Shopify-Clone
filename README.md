## **BreakPoints** :

I have taken five breakpoints to make it responsive.

-   sm = **640px**
-   md = **768px**
-   lg = **1024px**
-   xl = **1280px**
-   2xl = **1536px**

## **Installation** :

1. Clone the repo

```
   git clone https://github.com/sonicDev99/Shopify-Clone.git

```

2. Install NPM packages

```
   npm install
```
